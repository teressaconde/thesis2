About System

Learn more about the Oral Fluency Classification platform, its purpose, workflow, and core capabilities.

System Overview

Oral Fluency Classification is a machine learning-powered system that uses Support Vector Machine (SVM) based models to automatically assess oral fluency from uploaded speech recordings. The system analyzes acoustic features to classify fluency levels and provides insights through interactive results and analytics dashboards.

Purpose

To provide an automated, objective, and reliable tool for evaluating oral fluency using advanced machine learning techniques.

Target Users

Researchers, educators, and students involved in speech assessment, language studies, and related academic research.

Version

v1.0.0

Initial release of the SVM-Based Fluency Assessment System.

About the Thesis / Research Background

This system is developed as part of an academic research study that investigates the effectiveness of SVM-based approaches in assessing oral fluency. It compares baseline and proposed SVM models using acoustic features such as MFCCs to provide accurate and consistent fluency classification.
